# install:

`git clone repo`<br>
`cd into new folder`<br>
`npm install` (installs express dependencies)<br>
`npm run client-install` (installs react dependencies)

# populate database:
located in `client/src/db/dbseed.sql`

# run as dev:

`npm run dev`
